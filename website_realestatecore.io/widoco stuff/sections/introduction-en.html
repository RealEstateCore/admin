What is RealEstateCore? back to ToC

RealEstateCore is the common language that will enable control over building and development of new services – the enabler of the promises of the digital transformation. A domain ontology preparing the buildings to interact with the Smart City made by and for property owners.

What are the benefits for the property owners?

Property owners use the RealEstateCore ontology to describe the data of interaction within the buildings that they operate, and the management, storage, and sharing of this data. More specific, the collection of different metadata schemes, its interconnection, and uses to describe buildings, inhabitants, events etc. The RealEstateCore focuses on merging and bridging three domains:
  • Digital representation of the building’s elements (e.g. BIM)
  • Control and operation of the building (e.g. Haystack, Ashrae standards)
  • Emerging IoT technologies (e.g. SSN, WoT, IPSO)

One more standard?

RealEstateCore is not a new standard. RealEstateCore uses and maps present standards in a pragmatic manner – a minimum viable design. Mappings to new standards are easy done by the user by adding an annotation (and possible shared with the community).
Note for this version: The mappings to other standards are annotations (e.g. "IFCComparable") and will be added to this documentation i upcoming revision

The RealEstateCore consist of a set of modules and controlled vocabularies

The moduls describe the specifics for a building like core (time, units etc.), building structure, and devices.
The Schemas are the controlled vocabulary (similar to tags in Haystack) that enables the interoperability of data (e.g. that temperature data collected from different systems has the same name).

Openness – Share the knowledge in order to scale

A consortium is formed with Vasakronan AB, Akademiska Hus AB, Klipsk AB, and Willhem AB as the founders and main sponsors.
The purpose of the consortium is to create an environment for cooperation and sharing of knowledge between property owners and partners.

How To Contribute?

We encourage you to contribute to make the RealEstateCore better. Please point out bugs or peculiarities, add or extend modules and vocabularies, suggest improvements. Please use our Github repo.
