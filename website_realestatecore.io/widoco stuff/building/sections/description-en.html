RoomType

In current version, only RoomType is defined in the Building ontology. A user can modify or create a new a list of needed room types. Other types of contrlled vocabularies needed to describe building parts are recommended to place in the Building ontotlogy.

RoomType is an example of a list of predefined instances that constitues of a controlled vocabulary.
